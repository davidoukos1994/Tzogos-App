# Live Songbook

Προσωπικό songbook για iPhone, με καλλιτέχνες, τραγούδια, στίχους/συγχορδίες, transpose και auto-scroll.

## Τρέξιμο τοπικά
```bash
npm install
npm run dev
```

## Hosting
Ανέβασμα σε Vercel ή Netlify:
- Build command: `npm run build`
- Publish directory: `dist`

## Χρήση
- Πρόσθεσε τραγουδιστή
- Πρόσθεσε τραγούδι
- Γράψε συγχορδίες σε μορφή `[Am]στίχος [G]στίχος`
- Άνοιξε Live mode για auto-scroll και transpose

Σημείωση: Για native Apple Watch app χρειάζεται ξεχωριστό SwiftUI/watchOS project. Αυτό είναι hosted web app/PWA για iPhone.
